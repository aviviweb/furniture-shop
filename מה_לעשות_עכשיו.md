# 🚨 Service CRASHED - מה לעשות עכשיו?

## ⚡ צעדים מהירים (2 דקות):

### 1. בדוק את ה-Logs
- **API Service** → **"Logs"** (משמאל)
- **גלול למטה** - מה כתוב שם? (העתק את השגיאה)

### 2. פתח את המדריך
**פתח: `תיקון_CRASHED.md`**

שם יש:
- איך לבדוק Logs
- איך לתקן לפי השגיאה
- פתרונות לשגיאות נפוצות

---

## 🔍 השגיאות הנפוצות ביותר:

### "Cannot find module '@prisma/client'"
→ **פתרון:** תקן את ה-Build Command (ראה `תיקון_עכשיו.md`)

### "JWT_SECRET is required"
→ **פתרון:** הוסף `JWT_SECRET` ב-Variables

### "Cannot connect to database"
→ **פתרון:** וודא שיש PostgreSQL Service בפרויקט

---
